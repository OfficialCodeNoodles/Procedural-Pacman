Font Name:		Wobble Board

Original source:	Rolf Harris Picture Builder

Original platform:	Commodore 64

Publisher:		Commodore Software

Year:			1984


From a program where you could build pictures on a screen character by character using your joystick.  If you can think of Paintbrush on early versions of Windows, this would be its predecessor.


You can find some of my other work on Dafont or Fontstruct.  I'm also on Twitter @omegaville.
